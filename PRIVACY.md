# Privacy Policy

## Data Collection

This Kiro Power does **not** collect, store, or transmit any user data. It operates entirely within your local Kiro IDE session.


## What This Power Does

This power provides documentation-only guidance (steering files) for validating Amazon Data Firehose streaming workflows. It does not:

- Connect to any external services
- Send telemetry or analytics
- Access your AWS credentials or account data
- Store any information between sessions

## AWS API Calls

Any AWS API calls made while using this power's guidance are initiated by you through your own AWS credentials and Kiro IDE. This power does not make API calls on your behalf.
